# 批量操作标签选择交互改进

## 改进内容

将 `BatchActionsTab.vue` 的标签选择从简单下拉框改为网格化多选界面，参考了 `LabelSelector.vue` 的设计模式。

## 主要变化

### UI改进
1. **搜索功能**：添加了搜索框，支持按标签名过滤
2. **网格展示**：标签以卡片形式网格排列，替代下拉选择框
3. **多选支持**：可同时选择多个标签进行批量操作
4. **视觉反馈**：选中标签有明显的视觉高亮
5. **快捷键**：支持数字键1-9快速切换标签

### 交互改进
1. **所见即所得**：无需打开下拉框就能看到所有标签
2. **批量选择**：一次可选择多个标签执行操作
3. **清晰反馈**：已选标签以Tag形式显示，支持单独移除
4. **快捷操作**：快捷键提升操作效率

### 代码变化
1. **组件结构**：从简单select改为复杂的搜索+网格+选中展示结构
2. **事件参数**：emit参数从单个标签改为标签数组
3. **状态管理**：添加了selectedLabels和labelSearch状态
4. **事件处理**：完整的键盘事件监听和处理

## 影响范围

- `BatchActionsTab.vue` - 主要改动
- `FilterPanel.vue` - 事件类型更新
- `BatchAnnotationPage.vue` - 事件处理方法更新

## 用户体验提升

1. **效率提升**：一次可选择多个标签，减少重复操作
2. **视觉友好**：网格布局更直观，减少认知负担
3. **快捷操作**：键盘快捷键支持提升专业用户体验
4. **搜索便利**：标签多时可快速搜索定位

## 技术细节

- 保持响应式设计，网格自适应
- 添加loading状态和空状态处理
- 键盘事件在组件挂载/卸载时正确绑定/解绑
- 操作完成后自动清空选择状态
